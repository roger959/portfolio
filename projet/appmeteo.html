🌦️ Application Météo – Projet individuel

📜 Description du projet

Dans le cadre de ma formation, j’ai développé une application météo en autonomie. L’objectif était de proposer une interface simple et intuitive permettant d’obtenir la météo actuelle dans tous les pays du monde. Grâce à la géolocalisation, l’utilisateur peut également voir la température et les conditions météorologiques de son environnement immédiat.

Cette première capture présente l'ensemble de l'application météo avec son interface principale et l'organisation globale des informations affichées à l'utilisateur.

Cette deuxième capture montre le rendu obtenu lorsque l'utilisateur autorise la géolocalisation pour afficher la météo correspondant à sa position actuelle.

Cette dernière capture illustre l'affichage météo après la sélection d'un lieu au hasard, afin de montrer que l'application peut aussi être utilisée sans géolocalisation.

🔧 Technologies utilisées

  • HTML – pour la structure de la page
  • CSS – pour le design responsive et agréable
  • JavaScript – pour interagir avec l’API météo et gérer la géolocalisation
  • API OpenWeatherMap – pour récupérer les données météorologiques en temps réel

📍 Fonctionnalités principales

  • Recherche météo par ville ou pays
  • Affichage de la température actuelle, du ressenti, de l’humidité et du vent
  • Utilisation de la géolocalisation pour afficher la météo autour de l’utilisateur
  • Interface responsive et adaptée aux mobiles

💡 Objectifs pédagogiques

Ce projet m’a permis de mieux comprendre le fonctionnement des API REST en JavaScript, la gestion des permissions de géolocalisation dans le navigateur, et d’améliorer mes compétences en structuration HTML/CSS moderne.

🛠️ Compétences développées

  • Manipuler des données JSON issues d’une API
  • Utiliser navigator.geolocation pour détecter la position de l’utilisateur
  • Créer une interface claire avec des données en temps réel
  • Travailler de manière autonome sur toutes les étapes du développement

🎓 Conclusion

L’application météo m’a permis d’aborder un sujet concret, utile et facile à tester. Ce projet m’a donné davantage de maîtrise des API externes et des interactions dynamiques avec l’utilisateur.
